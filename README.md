# Conveyor Redux

> Conveyor helper for working with Redux stores.

[![NPM Version][npm-image]][npm-url]
[![Build Status][travis-image]][travis-url]
[![Downloads Stats][npm-downloads]][npm-url]

Conveyor Redux is a JavaScript library that works with Conveyor to generate reducers, epics, and actions for Conveyor projects using Redux. It automates the process of creating actions, reducers, and epics that are needed for Conveyor.

![](header.png)

## Installation

OS X & Linux:

This project is dependent on Conveyor, if you don't already have Conveyor use the following command to install both Conveyor and Conveyor Redux.

```sh
yarn add conveyor conveyor-redux
```

If you already have Conveyor installed just remove Conveyor from the previous command.

If you use npm use the following command.

```sh
npm install conveyor conveyor-redux
```
